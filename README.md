OSC controler are connectable to all major music production programms, including Ableton, FL Studio and Bitwig Studio.

Open Stage Version used: 1.30.2
To use this controler you need Open Stage Control ( and a midi port emulator. I recommend loopMidi, its free and i tested this controler with loopMidi. But other software could work too. You have to define 2 midi-ports (in a software you chosen): OSC-IN and OSC-OUT.

To use the control buttons like 'to start' or 'line up' you have to use a midi to key 'transformator' software. If you use ableton Suite you have max4Life included. If you dont have it i recommend Key2Key, its free aswell.

General Controler: includes 2 modes: Pad and Fader mode. You can switch between them easily. You can define own themes or use the pre-selection. Theme options: red, pink, blue, turquoise, green, yellow and custom with rgb-faders. Also the General Controler automaticly addapts to your window size.
